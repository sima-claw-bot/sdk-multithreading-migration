## What This Task Does
Generates a DotNetCliTool XML settings file at `ToolsSettingsFilePath`. Contains command info, entry point, runner, and optional RuntimeIdentifierPackages.

## Forbidden API Usage Found
```csharp
protected override void ExecuteCore()
{
    GenerateDocument(...).Save(ToolsSettingsFilePath);
}
```
- `ToolsSettingsFilePath` — passed directly to `XDocument.Save()`, could be relative path

## Migration Required
1. Add `[MSBuildMultiThreadableTask]` attribute
2. Implement `IMultiThreadableTask`
3. Absolutize `ToolsSettingsFilePath` before passing to `.Save()`

### Expected Change
```csharp

    GenerateDocument(...).Save(settingsPath);
}
```
